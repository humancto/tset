# Code of Conduct

This project follows the [Contributor Covenant v2.1][cc].

In short: be kind, be specific, focus on the work. Harassment of any kind is
not tolerated.

Reports go to the maintainers listed in [`governance/GOVERNANCE.md`](governance/GOVERNANCE.md).

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
